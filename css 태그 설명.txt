직접 css 수정해보면서 깨달은 것들
1. padding : 그냥 각 html 태그들 간의 공백 정하는 거
2. margin : header, section, footer 같은 container들 간의 공백 정하는 거
3. display: flex; : 웹페이지 화면 크기에 맞게 html 태그의 배치를 알아서 조정
4. flex-direction : 배치 조정할 방향 (상하, 좌우) 설정해줌
5. font-size : 글자 크기 설정
6. font-weight : 글자의 굵기 설정
7. list-style-type : li 태그의 순서 나타내는 모양 설정
8. text-decoration : 글자의 모양(밑줄 등) 설정
9. @media screen and (조건) : 웹페이지 화면이 조건과 같아지면 media screen문 아래의 css를 적용
10. text-align : 본문의 정렬방향 설정
11. border : 태그의 윤곽선 설정
12. border-radius : 윤곽선 모서리를 둥글게 설정
